# Changesets

Hello and welcome! This folder has been automatically generated by [`@changesets/cli`](https://github.com/changesets/changesets).

## Creating a changeset

```bash
pnpm changeset
```

Pick the packages you changed, pick a bump type (patch / minor / major), write a one-line summary that will appear in CHANGELOG.md.

## Releasing

The GitHub Action in `.github/workflows/release.yml` watches for pushes to `main`:

- If any `.changeset/*.md` files exist, the action opens a **"Version Packages"** pull request that bumps versions and updates changelogs.
- Merging that PR triggers the publish step, which runs `pnpm publish -r` to push the bumped packages to npm.

For local releases:

```bash
pnpm changeset version     # bump versions + write CHANGELOG.md
pnpm changeset publish     # publish bumped packages to npm
```
